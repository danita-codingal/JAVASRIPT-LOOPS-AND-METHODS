SWITCH STATEMENT IN JAVASCRIPT

In this activity, learn about the switch statement and how it is useful in JS

The switch statement is used to perform different actions based on different conditions. It is similar to a series of if...else statements but is more concise and easier to read when you have multiple conditions to check.

Here is the syntax of a switch statement:


        switch(expression) {
            case value1:
                // code to be executed if expression === value1
                break;
            case value2:
                // code to be executed if expression === value2
                break;
            ...
            default:
                // code to be executed if expression doesn't match any case
        }
